### AIEditor is a smart, distraction-free writing tool that helps you create, improve, and extend your content instantly. Just write, click “Continue Writing,” and let AI generate clean, coherent text that matches your style.
### To run this project follow the following steps
  1. clone this repo in you pc
  2. create gemini api key and paste in .env file, 
  [ Get Gemini API key](https://ai.google.dev/gemini-api/docs/api-key)
 
     
  3. run following command in terminal
       ```
       npm install
       npm run dev
       ```
